Penggunaan

Untuk Menggunakan End Point, Kirim Sebuah Permintaan GET kepada End Point yang Diinginkan Dengan Diikuti Parameter Berikut :

    Parameter  => text 

Contoh:
https://contoh.com/luminai?text=
04

Respon

Respon Yang Diberikan Berupa Format JSON Dengan Struktur :

{
  "status": 200,
  "creator": "AP",
  "data": {
    "response": "Hasil Respon Permintaan Kamu"
  }
}
